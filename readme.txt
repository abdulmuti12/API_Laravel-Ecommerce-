petunjuk Running Aplikasi

1.clone project
2.masuk ke project laravel lalu klik composer install diterminal
3.buat database dengan nama "mini-ecommerce"
4.untuk database bisa di import atau di migrasi
 a.untuk migrasi bisa tuliskan cmd "php artisan migrate" di terminal vscode
 b.untuk import database bisa pake file yang ada di folder sql dengan nama mini-ecommerce.sql
5.jika database sudah selsai di import atau di migrasi bisa tambahakan 
env dengam mengubah file example_env.txt jadi .env dan sesuaikan username dan password db di env sesuaikan dengan database di local
6.jika sudah selsai tinggal ketika "php artisan serve" di terminal maka api laravel backend ecommerce sudah jalan.
